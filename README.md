# Digital Literacy Project

## 📌 Overview
This project focuses on developing essential digital skills for students, including digital literacy, online safety, communication, and professional online presence. It combines practical tasks and tools to enhance academic and career readiness.

## 📝 Project Structure

### Task 1 – Infographic
- Designed using Canva
- Covers digital literacy, useful tools, and safe internet practices
- Focus on visual communication and clarity

### Task 2 – Professional Profiles
- Created and updated profiles on:
  - GitHub
  - LinkedIn
  - Kaggle
- Aim: Build a strong academic and professional presence

### Task 3 – Tools & Collaboration
- Practiced coding on HackerRank
- Created a Google Form quiz
- Used Google Sheets for response collection

### Task 4 – Email Etiquette & Social Media
- Wrote professional emails (extension request & internship inquiry)
- Learned responsible social media usage
- Understood consequences of poor digital communication

### Task 5 – Cybersecurity Awareness
- Case study on phishing attacks
- Identified risks and preventive measures
- Developed safe online practices checklist

## 🎯 Learning Outcomes
- Improved digital literacy and research skills
- Gained experience with productivity and collaboration tools
- Enhanced professional communication
- Developed awareness of cybersecurity threats
